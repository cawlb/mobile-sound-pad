Mobile Sound Pad

Welcome to the Mobile Sound Pad project! This is a mobile-friendly sound pad application with multi-touch support and visual effects.

Download the Project

Due to GitHub file size limitations, the full project with all sound files couldn't be uploaded directly. To use this sound pad, please:

  1. Clone this repository: git clone https://github.com/cawlb/mobile-sound-pad.git
  2. Download the full source files from the link provided to you
  3. Copy the sound files into a folder named sounds in the project directory

Running the Sound Pad

After downloading the files, you can run the sound pad using a simple HTTP server:

# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# If you have Node.js
npx serve

Then access the application at:

  • On your computer: http://localhost:8000/soundpad.html
  • On your mobile device: http://YOUR_LOCAL_IP:8000/soundpad.html (replace YOUR_LOCAL_IP with your computer's local IP address)

Features

  • Simple, clean white grid interface
  • Multi-touch support for playing multiple sounds simultaneously
  • Sliding/swiping functionality across multiple pads
  • Full-screen visual effects that react to sounds
  • Different shapes and colors for each sound type
  • Visuals overlap when multiple sounds are played

For more details, please see the README.
